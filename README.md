#### CLI

```pwsh
$ npm i -g @ionic/cli
```

##### For web

```bash
$ ionic serve
```

##### For mobile

```zsh
$ ionic build
$ npx cap add android
$ npx cap add ios
$ npx cap open android
$ npx cap open ios
$ ionic capacitor run android --livereload --external
$ ionic capacitor run ios --livereload
```
